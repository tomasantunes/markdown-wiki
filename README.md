# markdown-wiki
Create a wiki with markdown files

### How to run app

```
Install Python3
pip install Flask markdown Flask-JSGlue
python main.py
On your browser go to localhost:5000/wiki
You can change your wiki's route, folder and title.
```

### How to update repo

```
python update.py
```

### How to update feed

```
python feed.py "Message"
```
